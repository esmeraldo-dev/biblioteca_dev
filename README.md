# 📚 Biblioteca DEV API

Este projeto é uma API REST robusta para o gerenciamento de uma biblioteca, desenvolvida como parte do meu portfólio de backend Java. O objetivo principal foi aplicar conceitos avançados do ecossistema Spring e garantir uma aplicação resiliente e escalável.

## 🚀 Tecnologias Utilizadas
* **Java 21** (Uso de Records e novas features)
* **Spring Boot 3**
* **Spring Data JPA**
* **PostgreSQL** (Persistência de dados)
* **Lombok** (Produtividade)
* **Bean Validation** (Validação de dados)
* **Flyway** (Migrations de banco de dados)

## 🛠️ Diferenciais Técnicos
* **Camada de Service:** Lógica de negócio isolada, garantindo integridade com `@Transactional`.
* **Tratamento de Erros:** Implementação de um Global Exception Handler para respostas HTTP padronizadas.
* **DTO Pattern:** Segurança e separação de responsabilidades entre a API e o Banco de Dados.
* **Clean Code:** Código escrito seguindo princípios de legibilidade e manutenibilidade.
